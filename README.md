# 🚀 Getting Started

## Prerequisites

Make sure you have installed:

* Flutter SDK
* Dart SDK
* Android Studio / VS Code
* Firebase Project Setup

---

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/magarohan/blood_link_flutter.git
```

## 2️⃣ Navigate to Project Directory

```bash
cd blood_link_flutter
```

## 3️⃣ Install Dependencies

```bash
flutter pub get
```

## 4️⃣ Run the App

```bash
flutter run
```

---

# 🔥 Firebase Setup

1. Create a Firebase project
2. Enable:

   * Authentication
   * Firestore Database
   * Storage
3. Download configuration files:

   * `google-services.json`
   * `GoogleService-Info.plist`
4. Place them in their respective directories

---

# 📦 Build APK

```bash
flutter build apk --release
```

APK location:

```bash
build/app/outputs/flutter-apk/app-release.apk
```
